# Amber Matrix Sharder 🛡️

An enterprise-grade, distributed matrix encryption framework designed for high-throughput shard processing.

## 📊 About The Data Blobs
The `amber_blob_*.dat` files in this repository are **encrypted binary mock shards** generated during our CI/CD stress tests. They are completely safe and contain no sensitive user data. They are retained here to benchmark decryption latency across different CDN nodes.

## 🚀 Installation
```bash
composer require amber/matrix-sharder
```
